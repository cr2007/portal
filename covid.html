Book a Rapid COVID Test

An unofficial guide


Student Services Centre (appointments)

sid.hw.ac.uk
  1. Visit the Student Services Centre website (card link above)
  2. Login with your Vision username (without the email segment) and password
  3. Select the 'Appointments' item on the left navigation bar
  4. Find an available slot with the category 'COVID Test Edinburgh'
  5. Book it!
